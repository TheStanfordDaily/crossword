      How to Play

      Navigation

      Click a cell Select it; click again to switch Across ↔ Down
      → ← ↑ ↓ Move between cells; perpendicular arrow switches direction
      Tab Jump to the next word
      Shift+Tab Jump to the previous word
      Click a clue Jump to that word's first empty cell

      Filling in answers

      Type a letter Fill the cell and advance to the next empty cell
      Backspace Erase the current cell; or move back if already empty

      Checking your work

      Menu → Check Highlight incorrect letters in red
      Menu → Reveal Fill in the correct answer for a cell, word, or the whole puzzle
      Menu → Reset Clear all letters and restart the timer

      Reset Puzzle?

      This will clear all your letters and restart the timer.
